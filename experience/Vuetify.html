# 主题

# 浅色深色切换

Vuetify支持Material Design规范的浅色和深色版本。 该名称始于根应用程序组件v-app，并得到大多数组件的支持。 默认情况下，应用程序将使用浅色主题，但是可以通过在主题服务中添加深色选项来轻松覆盖它。

// src/plugins/vuetify.js

import Vue from 'vue'
import Vuetify from 'vuetify/lib'

Vue.use(Vuetify)

export default new Vuetify({
  theme: {
    dark: true,
  },
})

# 显示

# 默认断点

设备 代码 类型 范围
特小 xs 手机 < 600px
小 sm 中小型平板 600px~960px
中 md 大型平板、笔记本 960px~1264px
大 lg 桌面电脑 1264px~1904px
特大 xl 4k和超宽 >1904px

# 设置

指定元素的display属性。 这些类可以应用于从xs到xl的所有断点。使用类.d-{value}时，等同为.d-${value}-xs。

  • .d-{value} 对应 xs
  • .d-{breakpoint}-{value} 对应 sm, md, lg and xl

其中value的属性是以下之一：

  • none
  • inline
  • inline-block
  • block
  • table
  • table-cell
  • table-row
  • flex
  • inline-flex

# 可见

根据当前视口有条件地显示元素。断点实用程序类始终从下至上应用。.d-none将应用于所有断点，而.d-md-none仅适用于md及更高版本。

屏幕大小 类 屏幕大小 类
全部隐藏 .d-none 全部可见 .d-flex
仅在 xs 隐藏 .d-none .d-sm-flex 仅在 xs 可见 .d-flex .d-sm-none
仅在 sm 隐藏 .d-sm-none .d-md-flex 仅在 sm 可见 .d-none .d-sm-flex .d-md-none
仅在 md 隐藏 .d-md-none .d-lg-flex 仅在 md 可见 .d-none .d-md-flex .d-lg-none
仅在 lg 隐藏 .d-lg-none .d-xl-flex 仅在 lg 可见 .d-none .d-lg-flex .d-xl-none
仅在 xl 隐藏 .d-xl-none 仅在 xl 可见 .d-none .d-xl-flex

此外，还可以使用横向显示帮助器类基于当前视口显示元素。 可以使用以下格式的hidden-{breakpoint}-{condition}应用这些类。 其中的condition在以下的基础上应用：

  • only: 仅在指定断点处隐藏元素
  • and-down: 在指定断点处及其以下隐藏元素
  • and-up: 在指定断点处及其以上隐藏元素

# 高程(Elevation)

高程帮助器可以控制沿z轴的两个曲面之间的相对深度或距离。 共有25个海拔高度。 您可以通过使用elevation-{n}类来设置元素的海拔高度，其中n是0-24之间与所需海拔高度相对应的整数。

# Flex 增长和收缩

Vuetify具有用于手动应用增长和收缩的帮助程序类。 可以通过添加格式为flex- {condition}-{value}的帮助类来应用这些条件，其中condition可以grow或shrink，value可以是0或1。grow将允许元素增长以填充可用空间，而shrink将允许元素缩小到仅其内容所需的空间。 但是，只有在元素必须收缩以适合其容器（例如调整容器大小或受flex-grow-1影响）时，才会发生这种情况。值0将阻止条件的发生，而值1将允许条件。 提供以下类：

  • flex-grow-0
  • flex-grow-1
  • flex-shrink-0
  • flex-shrink-1

这些帮助类也可以以flex-{breakpoint}-{condition}-{state}格式应用，以基于断点创建更具响应性的变体。可以使用以下组合：

  • flex-sm-grow-0
  • flex-md-grow-0
  • flex-lg-grow-0
  • flex-xl-grow-0
  • flex-sm-grow-1
  • flex-md-grow-1
  • flex-lg-grow-1
  • flex-xl-grow-1
  • flex-sm-shrink-0
  • flex-md-shrink-0
  • flex-lg-shrink-0
  • flex-xl-shrink-0
  • flex-sm-shrink-1
  • flex-md-shrink-1
  • flex-lg-shrink-1
  • flex-xl-shrink-1

# 间距

更新布局而不创建新类。 Spacing帮助器对于修改元素的填充和边距很有用。

# 使用方法

帮助类将边距或填充应用于范围从0到16的元素。可以使用以下格式{property}{direction}-{size}来应用这些类。

该属性应用间距类型：

  • m: margin
  • p: padding

方向指定该属性应用于的一侧:

  • t: *-top
  • b: *-bottom
  • l: *-left

# 程序滚动

您可以使用$vuetify对象上的goTo方法以编程方式触发应用程序中的滚动。 此方法支持几种不同类型的目标选择器和选项，包括使用内置缓动功能的平滑滚动。

# 使用方法

goTo方法采用两个参数target和options。 target可以是距页面顶部的像素偏移量、有效的CSS选择器或元素引用。 options是一个包含duration，easing，container和offset的对象。

# 指令

# 外部点击（Click outside）

当单击目标元素之外的某物时，v-click-outside指令将调用一个函数。 v-menu和v-dialog等组件在内部使用此功能。

# 路口观测者（Intersection observer）

v-intersect指令利用了Intersection Observer API。 它提供了一个易于使用的界面，用于检测元素在用户视口中何时可见。 这也用于v-lazy组件。

# 变动观测者（Mutation observer）

v-mutate指令利用了Mutation Observer API。 它提供了一个易于使用的界面来检测元素何时更新。

# 调整大小指令

当窗口调整大小时，可以使用v-resize指令调用特定的函数。

# 波纹指令

v-ripple指令用于显示用户的操作。 它可以应用于任何块级元素。 内置的涟漪指令附带了许多组件，例如v-btn，v-tabs-item等。

# 滚动指令

当滚动窗口，指定的目标或元素本身（带有.self修饰符）时，可以使用v-scroll指令提供回调。

# 触摸支持

v-touch指令允许您捕获滑动手势并应用定向回调。
